Form Intro

Form => Form Tag is USed to make a Form in HTML

action => Path Backend

methodTwo types of Method

  • get => less secure, shows data in URl, easily Bookmarked, Limitation
  • post => secure, Never shows data in URl, Not Bookmarked, No Limitation, used for file upload

enctype

  • application/x-www-form-urlencoded => By Default
  • multipart/form-data => When input type is file
  • text/plain => used for Debugging process
