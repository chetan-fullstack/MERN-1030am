To indicate that an HTML document or element is in Spanish, use the lang attribute with the value es. For example, to specify the language of the entire document, you would use <html lang="es">. To specify the language of a specific paragraph, you would use <p lang="es">This is a Spanish paragraph</p>. [1, 2, 3, 4] Detailed Explanation: Mi Página en Español
